Structure of the dataset¶

  • The dataset has a total of 16 features and 183,412 trips
  • six of the columns contain null values
  • There's no duplictaes in the data set
  • The data set mainly has three data types, object, float and int
  • The start_time and end_time features of this data types are objects, they need to be coverted to datetime data type

Main feature(s) of interest in the dataset¶

  • the main features we will look at in this data set are:
    • duration_sec
    • user_type
    • start_time

Features in the dataset that will help support the investigation into the feature(s) of interest¶

  • Features like end_time, start_station_name, end_station_name will help us to determine the time and place where trips are taken.
  • The user_type and membe_gender features will also help us to determine the type of users and genderof the people who uses the bikes often.

Conclusions¶

The dataset contains data of bike rides taken on the moth of February and March in 2019

Most trips were taken between Monday and Frday with the highest number of trips occuring on Thursdays and Tuesdays, the weekend has fewer number of trips taken.

The rush hours; going to work in the morning (8 and 9 hrs) and closing in the evening (17 and 18 hrs) have the highest trip records

About 90% of the trips were taken by subscribers

Most of the trips were taken by males (>70%)

Customers spend more duration than subscribers

Customer type doesn't really affect the number of trips per weekday

Customers travel for longer durations regardless of the day of the week as compared to subscribers.
